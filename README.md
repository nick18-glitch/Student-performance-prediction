# Student Performance Prediction

This project uses a **Machine Learning regression model** to predict student performance based on factors such as study time, previous scores, and other academic features.

## Dataset

The dataset contains information about students including study hours, previous scores, and other attributes that influence academic performance.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## Steps Performed

* Data cleaning and preprocessing
* Feature selection
* Train-test split
* Model training and prediction

## Model Evaluation

The model performance was evaluated using metrics such as:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

## Goal

To understand the basics of **regression models and predicting numerical outcomes using machine learning**.
